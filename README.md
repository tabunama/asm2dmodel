# asm2dmodel

Python package for ASM2d compiled extension modules.

## Installation

Install from PyPI:

```bash
pip install asm2dmodel
```

## Import

```python
from asm2dmodel import asm2d, combiner, carboncombiner, metalcombiner, hyddelay, settler1d, primclar
```

## Current modules

- `asm2d`
- `combiner`
- `carboncombiner`
- `metalcombiner`
- `hyddelay`
- `settler1d`
- `primclar`
